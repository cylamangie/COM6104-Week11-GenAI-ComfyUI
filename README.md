# Generative AI with ComfyUI – Week 11

## 📌 Overview
This project explores **ComfyUI**, a node-based interface for generative AI image creation.  
It demonstrates how to build workflows, use ControlNet for pose guidance, and integrate prompts for creative image generation.  
Due to time and environment constraints, only screenshots and related notebooks (Week 5 and Week 6) are included here.

> 📝 **Note**: This is the **eleventh lab assignment** for **COM6104 – Topics in Data Science and Artificial Intelligence**.

---

## 🎯 Motivation
ComfyUI provides a **visual workflow editor** for diffusion models.  
This lab highlights:
- How to install and run ComfyUI.  
- How to build workflows manually and from JSON.  
- How to use **ControlNet** for pose-guided image generation.  
- How to integrate workflows with Python scripts.

## 🚀 How to Run
Since JSON workflows are not included, this repo serves as **documentation only**.  
To reproduce:
1. Install ComfyUI locally.  
2. Download a diffusion model (e.g., PonyXL, ControlNet).  
3. Rebuild workflows manually using the screenshots as a guide.
